angular-flipsnap
A flipsnap.js directive for AngularJS

Getting Started

Installation

You can install this module using Bower (which will also install the dependencies). Include the script and declare the flipsnap module as a dependency of your app. 

bower install angular-flipsnap --save
angular.module('app', ['flipsnap', ...]);

This module requires AngularJS 0.2.x and Flipsnap.js 0.6.x.

Usage

You can place a flipsnapper in your code using the flipsnap directive. You must specify a name, an id, a repeater and a width for the flipsnapper.

Template 

<div flipsnap="flipsnap" width="320" repeat="item in items" id="flipsnapper1">
    <div>
        <h1>{{item}}</h1>
    </div>
</div>

Controller 

$scope.items = ['1', '2', '3'];

In this example, the flipsnap control exists on the $scope as $scope.flipsnap. The repeat expression is a simplified form of the ng-repeat expression which only supports arrays and the 'in' operator. The width expression is the desired width of the flipsnapper viewport in pixels.
